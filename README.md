# 🐶 TinDog - Dog Dating Website

TinDog is a fun and responsive website inspired by Tinder, but for dogs 🐕.
It allows dogs (and their owners 😄) to connect, find matches, and explore premium plans.

---

## 🚀 Features

* 🐾 Beautiful landing page design
* 📱 Fully responsive (works on mobile & desktop)
* 🎯 Easy navigation with Bootstrap navbar
* 💬 Testimonials carousel
* 💰 Pricing section with different plans
* 📥 Call-to-action download buttons
* 🌐 Social media footer

---

## 🛠️ Technologies Used

* **HTML5** → Structure of the website
* **CSS3** → Styling and layout
* **Bootstrap 5** → Responsive design and components
* **Font Awesome** → Icons

---

## ▶️ How to Run the Project

1. Clone the repository:

   ```
    https://janendrasingh.github.io/Tindog/
   ```

2. Run the project:

   * Simply open `index.html` in your browser
